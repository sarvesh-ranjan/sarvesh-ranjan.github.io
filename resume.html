Research

Scalable Iterative Algorithm for Subspace Clustering: Convergence and Initialization

  • Designed and developed a new iterative algorithm that solve incomplete robust PCA problem at each iteration to make more efficient algorithm for large-scale data and proved monotone convergence property of the proposed algorithm.
  • To solve local convergence issue, proposed a practical initialization method and now try to prove the theoretical result of proposed initialization method.
  • Reported experimental result on both synthetic and real world dataset (MNIST).
  • Submitted at NIPS 2015.

Affinity Propagation for Solving Robust Subspace Clustering

  • Designed an efficient Affinity Propagation algorithm for solving Robust Subspace Clustering based on result of previous research.

Scalable Network Monitoring System

  • Designed and developed high-performance and storage-efficient index structure for high-speed traffic monitoring and storing system. The system endures 40G traffic while most of real IDC endure less than 10G.
  • Designed and implemented a 2 level multi-thread index structure using the Pthread and the asynchronous event driven I/O. The hierarchy of the index structure includes bloomfilter and sorted array list.
  • Implemented event driven multi-thread query system using the Pthread.
